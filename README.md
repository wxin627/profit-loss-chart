一个基于 React 和 D3.js 的期权损益图表可视化组件，用于展示期权交易的潜在盈亏情况。

## 功能特点

- 动态展示期权损益曲线
- 支持看涨期权和看跌期权
- 实时交互式提示框显示具体价格点的盈亏情况
- 清晰的盈亏分区显示（绿色表示盈利区域，红色表示亏损区域）

## 技术栈

- React
- TypeScript
- D3.js
